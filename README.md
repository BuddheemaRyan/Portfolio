# Buddheema Ryan — Personal Portfolio

A responsive personal portfolio website showcasing my skills, projects, and experience as an aspiring software developer.

## 🌐 Live Demo

[View Portfolio](https://buddheemaryan.github.io/Portfolio/)

## ✨ Features

- **Responsive Design** — Works seamlessly on desktop, tablet, and mobile devices
- **Dark / Light Mode Toggle** — Switch between themes with a single click
- **Animated Sections** — Smooth scroll-triggered animations powered by AOS
- **Typewriter Effect** — Dynamic text animation on the hero section
- **Skills Section** — Animated progress bars for each technology
- **Projects Showcase** — Cards with live demo and GitHub links
- **Contact Form** — Get in touch directly from the site

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Structure & markup |
| CSS3 | Styling & animations |
| JavaScript (ES6+) | Interactivity & logic |
| [Bootstrap 5](https://getbootstrap.com/) | Responsive layout & UI components |
| [Bootstrap Icons](https://icons.getbootstrap.com/) | Icon library |
| [AOS](https://michalsnik.github.io/aos/) | Scroll animation library |

## 🚀 Getting Started

No build tools or dependencies are required. Simply open the project in a browser:

1. **Clone the repository**
   ```bash
   git clone https://github.com/BuddheemaRyan/Portfolio.git
   ```

2. **Open `index.html`** in your preferred browser, or use a local development server:
   ```bash
   # Using VS Code Live Server extension
   # Right-click index.html → "Open with Live Server"
   ```
